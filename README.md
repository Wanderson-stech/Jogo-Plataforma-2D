# Jogo Plataforma 2D

Projeto desenvolvido na Unity utilizando C# para implementação de mecânicas de movimentação, interação e progressão em fases.

## Sobre o Projeto

Este projeto consiste em um jogo de plataforma 2D onde o jogador deve atravessar obstáculos, coletar itens e alcançar o objetivo de cada fase.

O desenvolvimento foi realizado utilizando a engine Unity e a linguagem C#, aplicando conceitos de programação orientada a objetos, gerenciamento de cenas, colisões e física 2D.

## Funcionalidades

- Movimentação horizontal do personagem
- Sistema de pulo
- Sistema de escalada
- Colisões com plataformas e obstáculos
- Sistema de coleta de itens
- Transição entre fases
- Controle de câmera
- Interface de usuário básica
- Sistema de vitória e derrota

## Tecnologias Utilizadas

- Unity
- C#
- Physics 2D
- Tilemap
- Animator
